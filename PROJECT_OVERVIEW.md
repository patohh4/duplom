## 🗄️ База Даних

### Таблиці

#### 1. **users** - Користувачі
```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT UNIQUE NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
)
```

#### 2. **mood_entries** - Записи Настрою
```sql
CREATE TABLE mood_entries (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    mood_level INTEGER NOT NULL CHECK (mood_level >= 1 AND mood_level <= 10),
    mood_emoji TEXT,
    note TEXT,
    entry_date DATE NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    UNIQUE(user_id, entry_date)
)
```

#### 3. **recommendations** - Рекомендації
```sql
CREATE TABLE recommendations (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    mood_range_min INTEGER NOT NULL,
    mood_range_max INTEGER NOT NULL,
    recommendation_text TEXT NOT NULL,
    emoji TEXT
)
```

## 🏗️ Архітектура

### Архітектурний Паттерн: **MVC + Repository Pattern**

```
┌────────────────────────────────────────────┐
│          Presentation Layer (UI)           │
│    MainWindow, UserPanel, MoodPanel,       │
│         StatisticsPanel (JavaFX)           │
└────────────────┬─────────────────────────┘
                 │
                 ▼
┌────────────────────────────────────────────┐
│        Business Logic Layer (Service)      │
│       MoodTrackerService                   │
│  - управління користувачами                │
│  - операції з настроями                    │
│  - вибір рекомендацій                      │
│  - розрахунок статистики                   │
└────────────────┬─────────────────────────┘
                 │
                 ▼
┌────────────────────────────────────────────┐
│     Data Access Layer (Repository)         │
│  UserRepository, MoodEntryRepository,      │
│     RecommendationRepository               │
│  - SQL запити                              │
│  - маніпуляція даними                      │
└────────────────┬─────────────────────────┘
                 │
                 ▼
┌────────────────────────────────────────────┐
│      Database Connection Layer (JDBC)      │
│         DatabaseManager                    │
│  - ініціалізація драйвера                 │
│  - управління підключенням                 │
│  - ініціалізація таблиць                   │
└────────────────┬─────────────────────────┘
                 │
                 ▼
┌────────────────────────────────────────────┐
│      Persistence Layer (Database)          │
│      SQLite (mood_tracker.db)              │
│  - users                                   │
│  - mood_entries                            │
│  - recommendations                         │
└────────────────────────────────────────────┘
```

### Принципи Дизайну

- ✅ **Separation of Concerns** - відділення層
- ✅ **Repository Pattern** - абстракція БД
- ✅ **Service Layer** - бізнес-логіка
- ✅ **Exception Handling** - обробка помилок
- ✅ **Logging** - логування операцій
- ✅ **SOLID Principles** - одноповинність класів

## 🚀 Запуск Проекту

### Вимоги

- Java 17 або вище
- Maven 3.9 або вище
- 50 MB вільного місця на диску

### Встановлення

```bash
# 1. Клонувати або завантажити проект
cd /Users/stasrusnak/Desktop/duplom

# 2. Встановити залежності (перший раз)
mvn clean install

# 3. Скомпілювати проект
mvn clean compile
```

### Запуск Застосунку

```bash
# Спосіб 1: Через Maven
mvn javafx:run

# Спосіб 2: Через exec plugin
mvn exec:java -Dexec.mainClass="com.moodtracker.MoodTrackerApp"

# Спосіб 3: Запустити jar файл (після упаковки)
mvn clean package -DskipTests
java -jar target/mood-tracker-app-1.0.0.jar
```

### Упаковка в JAR

```bash
# Створити виконуваний JAR з залежностями
mvn clean package assembly:single

# Запустити jar
java -jar target/mood-tracker-app-1.0.0-jar-with-dependencies.jar
```
