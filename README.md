# MindDoc — Застосунок для психологічного самомоніторингу

Настільний застосунок для психологічного самомоніторингу з відстеженням настрою, журналюванням симптомів, КПТ-курсами та аналітикою.

## Функціонал

- **Трекер настрою** — щоденні записи рівня настрою з контекстом та нотатками
- **Журнал симптомів** — фіксація симптомів із зазначенням інтенсивності
- **КПТ-курси** — каталог когнітивно-поведінкових курсів із відстеженням прогресу
- **Вправи** — психологічні вправи з покроковими інструкціями
- **Аналітика** — динаміка настрою та статистика симптомів за обраний період
- **Налаштування** — вибір мови інтерфейсу, теми та розміру тексту

## Технологічний стек

- **Java 21** — мова програмування
- **JavaFX 21** — графічний інтерфейс
- **SQLite** — локальна база даних
- **Apache Maven** — система збірки
- **SLF4J / Logback** — журналювання

## Запуск

### Вимоги
- Java 21 або новіша версія

### Крок 1 — Перевірте версію Java
```bash
java -version
```

### Крок 2 — Зберіть проєкт
```bash
mvn clean package
```

### Крок 3 — Запустіть застосунок
```bash
java -jar target/mood-tracker-app-1.0.0.jar
```

> При першому запуску автоматично створюється база даних `mindoc.db` з усіма таблицями та початковими даними.

## Структура проєкту

```
src/main/java/com/mindoc/
├── database/       # Ініціалізація БД
├── model/          # Моделі даних
├── repository/     # Репозиторії (SQL-запити)
├── service/        # Сервісний рівень
├── ui/             # Панелі інтерфейсу
└── util/           # Утиліти
```
