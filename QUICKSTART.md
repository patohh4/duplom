# 🚀 Швидкий старт Mood Tracker

## 1️⃣ Перший запуск за 2 хвилини

### Встановлення залежностей
```bash
cd /Users/stasrusnak/Desktop/duplom
mvn clean install
```

### Запуск застосунку
```bash
# Варіант 1: Запуск з Maven
mvn clean javafx:run

# Варіант 2: Запуск JAR файлу (якщо вже упакований)
java -jar target/mood-tracker-app-1.0.0.jar
```

## 2️⃣ Основні кроки в застосунку

### Крок 1: Створіть користувача
1. Перейдіть на вкладку **"Користувачі"**
2. Введіть ім'я користувача (наприклад, "Іван")
3. Натисніть **"Створити"**
4. Натисніть **"Оновити"** щоб побачити списки

### Крок 2: Вберіть користувача
1. Натисніть на користувача в списку
2. Ви побачите повідомлення "Вибраний користувач: ..."

### Крок 3: Записуйте свій настрій
1. Перейдіть на вкладку **"Мій настрій"**
2. **Виберіть рівень** (1-10) за допомогою бігунка
3. **Виберіть емодзі** (опціонально)
4. **Додайте нотатку** (опціонально)
5. Натисніть **"Зберегти настрій"**

### Крок 4: Переглядайте рекомендації
- Після збереження запису ви побачите **рекомендації** для вашого настрою
- Рекомендації залежать від вибраного рівня

### Крок 5: Аналізуйте статистику
1. Перейдіть на вкладку **"Статистика"**
2. Натисніть **"Оновити"**
3. Побачите:
   - Середній настрій
   - Мінімальний/максимальний рівні
   - Графік за останні 14 днів

## 6️⃣ Розширені функції

### Переглянути історію
- На вкладці **"Мій настрій"**
- Натисніть **"Переглянути історію"**
- Побачите всі записи по датам

### Отримати статистику
- На вкладці **"Статистика"**
- Виберіть користувача та натисніть **"Оновити"**
- Графік оновиться автоматично

## 7️⃣ Problematic Scenarios (Розв'язання проблем)

### ❌ Помилка: "User not selected"
✅ **Рішення:** Спочатку оберіть користувача на вкладці "Користувачі"

### ❌ Помилка: "Cannot compile"
✅ **Рішення:** 
```bash
mvn clean install -U
```

### ❌ Графік не оновлюється
✅ **Рішення:** Натисніть кнопку "Оновити" на вкладці "Статистика"

### ❌ Запис не збережений
✅ **Рішення:** Перевірте, що введено значення для настрою (1-10)

## 8️⃣ Збереження та резервне копіювання

### Резервна копія БД
```bash
cp mood_tracker.db mood_tracker.db.backup
```

### Відновлення з резервної копії
```bash
cp mood_tracker.db.backup mood_tracker.db
```

## 9️⃣ Видалення записів (якщо потрібно)

⚠️ **Видалення користувача також видалить його записи!**

Перший видалити можна тільки через SQL:
```sql
DELETE FROM mood_entries WHERE id = 5;
```
